## Spring (dans /server)

Pour lancer le serveur
```
mvnw spring-boot:run
```

## Angular (dans /client)

Pour lancer le client
```
ng serve
```

Pour créer un nouveau component
```
ng g c nom-component
```

Pour créer un nouveau service
```
ng g s shared/nom-service/nom-service
```
